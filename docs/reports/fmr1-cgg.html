7. Interpretation

  • The reported FMR1 result (c.-129CGG, 29 repeats, Normal) concerns a locus that is outside the design footprint of this exome-plus-spike-in target BED.
  • On-target capture and per-base depth at the repeat cannot be inferred from this BED, because no probe interval overlaps chrX:147,912,050-147,912,110.
  • The repeat is GC-rich and expansion-prone; short-read exome data is not a validated method for sizing FMR1 CGG repeats regardless of tiling. The "29 / Normal" call should be traceable to a dedicated repeat assay.
  • If a coverage claim is required for this locus, depth must be read from an alignment/coverage file over the explicit interval, not assumed from panel membership.

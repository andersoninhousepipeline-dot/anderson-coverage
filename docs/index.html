Coverage Checker

Panel (BED) coverage + sample read-depth coverage · GRCh38 / hg38

An internal web tool to check NGS capture-panel coverage and real sample sequencing depth — by gene, multiple genes, chromosome region, transcript, or rs ID — with one-click branded PDF reports.

What each report shows

Panel coverage

Whether the gene / region / variant is in the capture design, targeted bp, and uncovered gaps — across any selected BED panel.

Sample read-depth

Real coverage from reference BAMs: mean ± SD, median, min, and % of target bases ≥ 1× / 10× / 20× / 30× / 50× / 100×.

Reference types

Normal, Male Infertility, Female Infertility, AF and POC — each aggregated across replicate samples, plus an Overall Coverage row.

Reports

Branded, print-to-PDF reports named after the query; downloadable matched intervals as BED.

This is a static information page. The Coverage Checker is a small Flask app. Reference coverage is shipped precomputed in coverage.db, so it runs straight from the repo — pip install -r requirements.txt then ./start.sh — no BAM files required. (GitHub Pages can't run the Python backend, so this page is informational only.) See the README for setup.
